# AI Quest

A Duolingo-inspired web app concept for learning AI tools and skills daily, with gamified progression and personal tutoring.

## Highlights

- Login setup with personalized user profile (username, email, role)
- Personal AI tutor guidance with refreshable daily lesson plans
- Role-based lesson personalization for executives, entrepreneurs, and builders
- Up-to-date lesson rotation including MCP servers and Openclaw concepts
- Streak tracking and reward loops (XP, gems, hearts, quests, chest)
- Curated links to external AI sources and platforms

## Run locally

1. Run a local server:
   - `python -m http.server 4173`
2. Open `http://localhost:4173` in your browser.
